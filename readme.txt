== Description ==

This plugin allows you to add tags to the header of an individual page or
post such as keywords/descriptions, custom meta tags, external style sheet
links, or embedded style tags. You no longer need to use conditional
statements or separate templates just to change something like a background
image, text style, etc.

Simply add your custom header into the fields of your post/page edit/create page. That's it!

== Installation ==

1. Upload `add_your_own_headers.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!
